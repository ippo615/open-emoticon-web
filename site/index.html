Open Emoticon
v0.0.1

A set of open source emoticon web fonts.

Download

Zip Archive Git hub page

Usage

Place the contents of the "build" folder with your website. For example:

site/
├── index.html
└── open-emoticon
	  ├── icon-utils.css
	  ├── open-emoticon.css
	  ├── open-emoticon.eot
	  ├── open-emoticon.html
	  ├── open-emoticon.ttf
	  └── open-emoticon.woff

In the head of your html:

<link rel="stylesheet" type="text/css" href="open-emoticon/icon-utils.css">
<link rel="stylesheet" type="text/css" href="open-emoticon/open-emoticon.css">

In the body of your html:

<i class="oe oe-winking-face"></i>
